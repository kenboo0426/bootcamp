# Stellar — 日本語ランディングページ

美しさ・速度・直感性を両立した、モダンな日本語ランディングページのテンプレートです。

## 特長
- レスポンシブ / ダークモード
- 日本語タイポグラフィ最適化（Noto Sans JP）
- セクション例（ヒーロー、特長、実績、価格、CTA）
- 軽量・高速、アニメーション最小限

## 使い方
1. 任意の静的サーバーで `index.html` を開きます。
2. 例（Python）：
   ```bash
   cd stellar-hp
   python3 -m http.server 5173
   ```
   ブラウザで `http://localhost:5173` を開きます。

## カスタマイズ
- 色やトーンは `styles.css` の CSS変数（:root）を書き換えてください。
- セクションの文言や順序は `index.html` を編集します。
- スクロール演出やテーマ切替は `script.js` で調整できます。
